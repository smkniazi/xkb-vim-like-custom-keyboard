# Vim Inspired Custom Keyboard Layout

`Caps + h`	--> 	`Left`
`Caps + j`	--> 	`Down`
`Caps + k`	--> 	`Up`
`Caps + l`	--> 	`Right`
`Caps + i`	--> 	`Home`
`Caps + o`	--> 	`End`
`Caps + p`	--> 	`Delete`
`Caps + e`	--> 	`Esc`
`Caps + d`	--> 	`Down`
`Caps + u`	--> 	`Up`
`Caps + m`	--> 	`Menu`

`Caps + [`	--> 	`å`
`Caps + {`	--> 	`Å`
`Caps + ;`	--> 	`ö`
`Caps + :`	--> 	`Ö`
`Caps + '`	--> 	`ä`
`Caps + "`	--> 	`Ä`
